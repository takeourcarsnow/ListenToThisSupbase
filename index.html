keyboard / tips
Welcome to ascii.fm!
  • Share and discover music links from YouTube, Spotify, Bandcamp, SoundCloud, or direct audio files.
  • Click [ login / register ] to create an account and start posting, liking, and commenting.
  • Use tags (e.g. #chill, #2020) to organize and filter posts.
  • Click [ play all ] to queue up the current feed and listen continuously.
  • All your data is saved in your browser (or synced with Supabase if enabled).
Posting
  • Fill in the title, artist, and a valid music link.
  • Add tags to help others find your post.
  • Optionally, write a short description or reason to listen.
Queue & Player
  • Use the player controls at the bottom to play, skip, shuffle, or repeat tracks.
  • Queue is built from the current feed (filtered by tag/search/sort).
Keyboard Shortcuts
  • / — focus search
  • n — focus composer
  • j / k — queue next / prev
  • l — like highlighted
  • o — toggle player on highlighted
  • ? — toggle this help
Tips
  • Click tags to filter the feed. Click [ clear tag ] to reset.
  • Export or import your data from the settings panel.
  • Everything is keyboard accessible for fast navigation.
